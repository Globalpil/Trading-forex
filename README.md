# Meta Forex Trading Platform

A comprehensive, professional Forex trading platform with advanced features including automated trading, multiple payment methods, and full administrative controls.

## 🌟 Features

### Core Platform
- **User Authentication** - Secure login system with admin and user roles
- **Admin Dashboard** - Complete control over users, balances, and platform operations
- **User Dashboard** - Personal trading space with performance analytics
- **Trading Terminal** - Professional trading interface with live price charts
- **Robot Trading** - Automated trading system with configurable parameters

### Trading Features
- **Multiple Currency Pairs** - EUR/USD, GBP/USD, USD/JPY, BTC/USD
- **Live Price Charts** - Real-time price updates with interactive charts
- **Buy/Sell Functionality** - Manual trading with leverage options
- **Stop Loss & Take Profit** - Risk management tools
- **Trade History** - Complete record of all transactions
- **Profit Tracking** - Separate real and profit balances

### Payment System
- **Multiple Deposit Methods**:
  - Bank Transfer (International & Local)
  - Payment Gateways (PayPal, Stripe, Flutterwave, Paystack)
  - Cryptocurrency (Bitcoin, Ethereum, USDT)

- **Withdrawal System** - Fast and secure withdrawals
- **Deposit Bonuses** - Tiered bonus structure

### Administrative Features
- **User Management** - Create, edit, activate, deactivate, and delete users
- **Balance Management** - Credit and debit user balances
- **Wallet System** - Real balance, profit balance, and credit balance
- **Robot Control** - Start/stop automated trading
- **Monitoring** - Real-time platform statistics
- **Transaction Approval** - Review deposits and withdrawals

### Security
- **Two-Factor Authentication** - Enhanced account security
- **KYC Verification** - Identity verification system
- **SSL Encryption** - Secure data transmission
- **Session Management** - Secure user sessions

## 🤖 Robot Trading System

The platform includes an automated trading robot with the following features:

### Configuration Options
- **Trading Mode**: Fully automatic, semi-automatic, or manual
- **Trade Duration**: Configurable in seconds (default: 6 seconds)
- **Profit Transfer Interval**: Set profit taking frequency (default: 2 seconds)
- **Max Trade Amount**: Limit per trade
- **Target Wallet**: Choose profit or real balance

### Trading Logic
- Monitors price movements for pull and fall patterns
- Automatically executes buy/sell orders
- Transfers profits to designated wallet
- 24/7 trading capability
- Configurable risk parameters

## 💰 Wallet System

### Balance Types
1. **Real Balance**: Primary trading funds
2. **Profit Balance**: Withdrawable profits from trades
3. **Credit Balance**: Bonus credits and rewards

### Withdrawal Rules
- All profit can be withdrawn
- All credit can be withdrawn
- No fees for withdrawals over $500
- 2% fee for withdrawals under $500
